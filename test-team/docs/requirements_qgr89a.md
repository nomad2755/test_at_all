## 三、业务规则

### 3.1 字段联动规则

1. **所在地区选择**：省-市-区三级联动
2. **地形地貌**：选择后显示（如"山地"）
3. **确认按钮**：点击后锁定当前录入信息，进入编辑状态

### 3.2 图片上传规则

1. **林木图片**：支持「手动上传」和「AI生成」两种方式
2. **林木不动产权证**：点击上传按钮添加
3. **产品详情图**：支持「默认图片」、「手动上传」、「AI生成」

### 3.3 AI 生成规则

1. 点击「AI生成」后显示「图片生成中」状态
2. 生成完成后显示「重新生成」和「确认图片」按钮
3. 支持对生成结果不满意时重新生成

### 3.4 批量录入规则

1. 支持批量录入多条林木信息
2. 每条记录独立，包含：林木图片、不动产权证、产品详情图
3. 批量模式下可多次添加

### 3.5 校验规则

1. **产权人与实名信息不一致**：提示"产权人与实名信息不一致，请重新上传"
2. **林权证示例**：提供林权证填写示例供参考

### 3.6 经纬度获取规则

> 通过高德地图或百度地图搜索 "东方市林业科学研究所"（地址：225 国道北侧、三月三广场西侧），长按地图位置即可查看经纬度

## 四、交互规则

### 4.1 录入流程

```
点击"添加林木" → 填写基础信息 → 确认 → 填写详细参数 → 上传/生成图片 → 提交
```

### 4.2 图片操作

| 操作 | 说明 |
|------|------|
| 手动上传 | 点击上传按钮选择本地图片 |
| AI生成 | 点击后自动生成，支持重新生成和确认 |
| 查看示例 | 可查看"林权证示例"作为参考 |

### 4.3 状态提示

| 状态 | 提示文案 |
|------|----------|
| 图片生成中 | "图片生成中" |
| 生成完成 | 显示"重新生成"和"确认图片"按钮 |
| 校验失败 | "产权人与实名信息不一致，请重新上传" |
