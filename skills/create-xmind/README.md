# 测试用例综合生成技能

## 快速开始

### 说一句话就能用

- "生成测试用例" → 生成 XMind 脑图 + 测试数据图片
- "生成详细测试用例" → 带前置条件、操作步骤、测试数据的 XMind
- "生成测试数据图片" → 生成证件票据图片用于 OCR 测试

---

## 两种核心能力

### 1. XMind 脑图生成

把测试用例整理成脑图，支持：

| 层级 | 内容 |
|------|------|
| 一级 | 功能模块（1. xxx 2. xxx） |
| 二级 | 功能描述 / 正向用例 / 负向用例 |
| 三级 | 具体用例（包含前置+步骤+数据+预期+验证） |

**生成方式**：
```
提供需求/原型 → 构建数据 → 生成 .xmind 文件
```

### 2. 测试数据图片生成

生成用于 OCR 识别测试的图片，支持：

| 类型 | 示例 | 生成方式 |
|------|------|----------|
| 中文证件票据 | 身份证、户口本、营业执照 | **PIL 渲染**（文字100%正确） |
| 文档模板 | 授权委托书、笔录表 | PIL 渲染 PDF 预览 |
| 风景素材 | 背景纹理 | MiniMax MCP |

## 典型使用场景

### 场景1：分析 Axure 原型，生成完整测试用例

```
你：分析这个原型并生成测试用例 https://xxx.com/原型页面
我：
  1. 打开原型截图分析
  2. 生成 XMind 脑图（8个模块，50+用例）
  3. 生成测试数据图片（7种证件）
  4. 输出到桌面
```

### 场景2：已有需求文档，生成 XMind

```
你：根据这个功能描述生成XMind脑图
    - 数据资产对接：DAT-01~DAT-05
    - 实名认证：AUTH-01~AUTH-08
我：生成 公证智能体四期测试功能点.xmind
```

### 场景3：生成特定测试数据图片

```
你：生成公证模块需要的测试图片
    - 身份证、户口本、执业证、任职证明
我：生成到 test_data/ 目录
    - 01_身份证.png
    - 02_户口本.png
    - 05_公证员执业证.png
    - 06_任职证明.png
```

## 重要提醒

- **中文证件票据**：只用 PIL 渲染，不要用 AI 生成
- **想要质感**：PIL 图片 + PS/Canva 手动叠加
- **XMind 格式**：XML（不是 JSON），兼容 XMind 2021
