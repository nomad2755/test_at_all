## 核心文件

`scripts/create_xmind.py`

## 关键结论

XMind 文件必须包含 **5 个 XML 文件**：

| 文件 | 说明 |
|------|------|
| content.xml | 主题内容 |
| styles.xml | 样式定义 |
| comments.xml | 批注 |
| meta.xml | 元数据 |
| META-INF/manifest.xml | 清单文件 |

> ⚠️ **注意**: 不能用 JSON 格式，必须用 XML
