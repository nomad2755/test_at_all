## 通用提示词要点

1. **文字必须清晰**：`all Chinese text crystal clear`, `high contrast`, `OCR-readable` — 每条提示词都要包含
2. **不要改变文字**：明确描述文字内容（车次、姓名、日期等），要求模型原样保留，不是重新生成
3. **风格描述**：描述质感、材质、布局，不描述具体中文文字的内容
4. **中文友好模型推荐**：通义万相、DALL-E 3（英文提示词）、Midjourney（英文提示词）、讯飞星火
